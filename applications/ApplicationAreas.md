## 📚 Application Areas

A non-exhaustive taxonomy of application areas where persona agents can drive evaluation, simulation, research, and experimentation. Each application area may contain multiple application types and concrete scenarios.

| Application Area                       | Application Type                  | Example Questions                                                                                                     |
| -------------------------------------- | --------------------------------- | --------------------------------------------------------------------------------------------------------------------- |
| **Product Research**                   | Product Concept Testing           | Would users understand and care about the product? Which positioning is most compelling?                              |
|                                        | Feature Prioritization            | Which proposed features matter most to different user segments? What should be built first?                           |
|                                        | Pricing & Packaging               | Would users purchase at this price point? Which pricing model or subscription tier provides the best perceived value? |
|                                        | Product-Market Fit                | Which user segments show the strongest demand? What unmet needs remain?                                               |
| **Market & User Research**             | User Segmentation                 | How do different user groups think, behave, and make decisions?                                                       |
|                                        | Pain Point Discovery              | What frustrations, unmet needs, and opportunities emerge?                                                             |
|                                        | Jobs-to-be-Done Analysis          | What goals are users trying to accomplish?                                                                            |
|                                        | Preference & Behavior Analysis    | How do preferences vary across personas and contexts?                                                                 |
| **Marketing & Advertising**            | Message Testing                   | Which messages and positioning strategies resonate most strongly?                                                     |
|                                        | Ad Evaluation                     | Which advertisements are most effective and memorable?                                                                |
|                                        | Brand Perception                  | How do users perceive the brand and its values?                                                                       |
|                                        | Campaign Evaluation               | Which campaigns drive awareness, engagement, and conversion?                                                          |
| **UX & Product Evaluation**            | Onboarding Evaluation             | Where do users get confused during onboarding?                                                                        |
|                                        | User Journey Analysis             | Which steps create friction or abandonment?                                                                           |
|                                        | Feature Discoverability           | Can users successfully find and use important features?                                                               |
|                                        | Accessibility Evaluation          | Can users with different abilities complete key tasks successfully?                                                   |
|                                        | Search & Discovery                | Can users find the information, products, or content they need?                                                       |
| **Conversational AI Evaluation**       | Assistant Evaluation              | Is the assistant helpful, trustworthy, and adaptive to different users?                                               |
|                                        | Task Completion Evaluation        | Can users successfully accomplish their goals with the assistant?                                                     |
|                                        | Multi-turn Dialogue Evaluation    | Can the assistant maintain context and coherence across interactions?                                                 |
|                                        | Persona Adaptation Evaluation     | Does the assistant appropriately adapt to different user backgrounds and needs?                                       |
| **Customer Support & Service**         | Issue Resolution                  | Can users resolve problems efficiently and successfully?                                                              |
|                                        | Escalation Handling               | Can difficult or frustrated users be handled appropriately?                                                           |
|                                        | Policy & Process Navigation       | Can users understand and complete support workflows?                                                                  |
|                                        | Satisfaction Evaluation           | Do users feel supported and satisfied with the experience?                                                            |
| **Recommendation Systems**             | Recommendation Quality Evaluation | Are recommendations relevant, useful, and personalized?                                                               |
|                                        | Diversity & Novelty Evaluation    | Do recommendations balance familiarity and discovery?                                                                 |
|                                        | Trust & Acceptance Evaluation     | Do users trust and accept the recommendations?                                                                        |
|                                        | Preference Alignment Evaluation   | How well do recommendations match user preferences?                                                                   |
| **Learning & Knowledge Transfer**      | Tutoring Evaluation               | Do users learn effectively from explanations and guidance?                                                            |
|                                        | Knowledge Assessment              | Can understanding and mastery be measured accurately?                                                                 |
|                                        | Learning Path Evaluation          | Does the system guide users through learning effectively?                                                             |
|                                        | Educational Content Evaluation    | How do users respond to instructional content?                                                                        |
| **AI Safety & Red Teaming**            | Adversarial Testing               | Can the system withstand malicious or manipulative behavior?                                                          |
|                                        | Privacy Evaluation                | Does the system handle privacy-sensitive interactions appropriately?                                                  |
|                                        | Jailbreak Evaluation              | Can safety policies be bypassed?                                                                                      |
|                                        | Harmful Content Evaluation        | Does the system avoid generating unsafe outputs?                                                                      |
| **Productivity & Workflow Assistance** | Workflow Optimization             | Can users complete tasks more efficiently?                                                                            |
|                                        | Knowledge Assistance              | Can users find and apply relevant information?                                                                        |
|                                        | Automation Evaluation             | Can repetitive tasks be automated effectively?                                                                        |
|                                        | Collaboration Support             | Does the system improve coordination and teamwork?                                                                    |
| **Community & Social Interaction**     | Engagement Analysis               | What drives participation and long-term engagement?                                                                   |
|                                        | Content Creation Evaluation       | How do users create and share content?                                                                                |
|                                        | Moderation Evaluation             | Are moderation policies effective and fair?                                                                           |
|                                        | Community Health Assessment       | How do interactions affect community quality?                                                                         |
| **Synthetic Data Generation**          | Dialogue Generation               | Generate realistic conversations between users and systems.                                                           |
|                                        | Preference Generation             | Generate preference datasets for personalization tasks.                                                               |
|                                        | User Journey Simulation           | Generate end-to-end behavioral trajectories.                                                                          |
|                                        | Behavioral Trace Generation       | Generate interaction logs and telemetry data.                                                                         |
