## 📚 Application Areas

A non-exhaustive taxonomy of application areas where persona agents can drive evaluation, simulation, research, and experimentation. Each application area may contain multiple application types and concrete scenarios.

| Application Area                       | Application Type                    | User-Centric Description                                                                                   |
| -------------------------------------- | ----------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| **Product Research**                   | Product Concept Testing             | How do different users understand, value, and respond to a new product concept?                            |
|                                        | Feature Prioritization              | Which features matter most to different user segments, and why?                                            |
|                                        | Pricing & Packaging                 | How do different users perceive pricing, value, and willingness to pay?                                    |
| **Market & User Research**             | User Segmentation                   | How do different user groups think, behave, and make decisions?                                            |
|                                        | Pain Point Discovery                | What frustrations, unmet needs, and motivations emerge across personas?                                    |
| **Marketing & Advertising**            | Message Testing                     | Which messages resonate with different user groups and why?                                                |
|                                        | Brand Perception                    | How do different personas perceive and trust a brand?                                                      |
| **UX & Product Evaluation**            | Onboarding Evaluation               | Which users struggle during onboarding, and where does friction occur?                                     |
|                                        | User Journey Analysis               | How do different personas move through a workflow, and where do they abandon tasks?                        |
|                                        | Accessibility Evaluation            | Which users face barriers when interacting with the product?                                               |
| **Conversational AI Evaluation**       | Persona Adaptation                  | How well does the AI adapt to users with different backgrounds, goals, and communication styles?           |
|                                        | Task Completion                     | Which users successfully accomplish their goals, and which encounter difficulties?                         |
| **Customer Support & Service**         | Issue Resolution                    | How effectively can different users resolve problems and obtain support?                                   |
|                                        | Escalation Handling                 | Which user types become frustrated, confused, or dissatisfied during support interactions?                 |
| **Recommendation Systems**             | Preference Alignment                | How well do recommendations align with the preferences of different users?                                 |
|                                        | Trust & Acceptance                  | Which users trust, accept, or reject recommendations, and why?                                             |
|                                        | Diversity & Exploration             | How willing are different personas to explore unfamiliar recommendations?                                  |
| **Learning & Knowledge Transfer**      | Learning Effectiveness              | How effectively do different users learn from explanations, feedback, and guidance?                        |
|                                        | Knowledge Assessment                | How do learning outcomes vary across user groups and learning styles?                                      |
| **AI Safety & Trust Evaluation**       | Privacy-Sensitive Users             | How do privacy-conscious users respond to the system's handling of personal information?                   |
|                                        | Vulnerable Users                    | How does the system influence users who may be easily misled, manipulated, or over-reliant on AI?          |
|                                        | Trust Calibration                   | Do different users develop appropriate levels of trust in the system's outputs?                            |
| **Productivity & Workflow Assistance** | Workflow Optimization               | Which users benefit most from workflow assistance, and where do bottlenecks remain?                        |
|                                        | Knowledge Assistance                | How effectively can different users find and apply relevant information?                                   |
| **Community & Social Interaction**     | Engagement Analysis                 | What motivates different users to participate, contribute, and remain active in communities?               |
|                                        | Moderation Evaluation               | How do different user groups perceive fairness, safety, and moderation decisions?                          |
| **Gaming & Interactive Experiences**   | Retention Analysis                  | Which player personas continue engaging over time, and which are likely to churn?                          |
|                                        | Progression & Difficulty Evaluation | How do different players respond to challenge levels, progression speed, and reward structures?            |
|                                        | Reward & Incentive Design           | Which rewards, achievements, and progression systems motivate different player types?                      |
|                                        | Social Play & Collaboration         | How do different personas interact with teammates, communities, and multiplayer systems?                   |
|                                        | Player Experience Evaluation        | How do different players perceive enjoyment, immersion, frustration, and satisfaction throughout gameplay? |
| **Synthetic Data Generation**          | User Journey Simulation             | Can realistic user behaviors, preferences, and interaction traces be generated across diverse personas?    |

### Design Principles

* **Persona-Centric:** Every application area should meaningfully depend on user characteristics, preferences, goals, or behaviors.
* **Cross-Domain:** Application areas are organized by evaluation objectives rather than industry verticals.
* **Scenario-Driven:** Each application type may contain multiple concrete scenarios across Survey, Chatbot, Web, and App environments.
* **Extensible:** New scenarios can be added without modifying the top-level taxonomy.
