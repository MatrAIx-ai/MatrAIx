What this PR does

Starclash is a crew game (2–4 players share one room, move, chat, and duel at Rock-Paper-Scissors for stars). The Playground samples one persona per trial, which doesn't fill a table on its own. This PR makes the crew game runnable as a single-persona Playground/Harbor web task by adding a player-vs-bots mode, plus a real vision/computer-use brain, a natural end-state, and per-persona batch reporting.

⚠ Not a real multi-agent game yet. In a trial, the one sampled persona is the real player; every other seat is a computer bot (BayesianBotBrain), not a second sampled persona. So a trial measures one persona's play against fixed bot opponents. Hosting several sampled personas together in one live match (true multi-agent) requires a Playground orchestration change — one trial that runs N persona agents in a shared game — which is not implemented here. Comparing persona styles today is done across separate sessions (swap the crew manifest), not within one live game.

Feature 2 — Real vision / computer-use brain

vision_brain.py: BrowserVisionBrain (--brain vision) renders each observation as an HTML HUD, loads it in a real headless Chromium via Playwright, screenshots it, and asks Claude (vision + forced tool-use) to click/type. The action is read back from the page's own window.__lastAction — genuine browser-use, not a JSON shortcut. Authenticates with a direct ANTHROPIC_API_KEY or an Amazon Bedrock bearer token (AWS_BEARER_TOKEN_BEDROCK + AWS_REGION), defaulting to Claude Sonnet (us.anthropic.claude-sonnet-4-5) on Bedrock.

Feature 3 — Natural end-state & per-persona reporting

  • engine.py: a game now ends the moment every active persona has emptied its hand (termination_reason: all_cards_played) — the intended end state — instead of burning the whole tick budget.
  • reporting.json / verifier: the verifier writes one task_outcome + one task_reasoning_trajectory context per crew persona, grouped by dominant_trait, so the batch report can compare outcomes and reasoning style across trait groups.
