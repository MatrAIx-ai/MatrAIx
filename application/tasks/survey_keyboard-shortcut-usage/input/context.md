# Keyboard Shortcut Usage

How people use keyboard shortcuts, hotkeys, macros, and related efficiency features across their software.

## About this survey

This survey measures how heavily people rely on keyboard shortcuts, which basic and advanced ones they use, and in which applications. It covers behaviors such as learning new shortcuts, remapping keys, using text expansion and command palettes, keyboard-versus-mouse balance, and specialized input hardware. It uses agreement-scale Likert batteries on productivity and frustration, a per-item proficiency matrix across browser, documents, spreadsheets, OS, and editors, a "top 3 reasons" ranking, and numeric free-text items on shortcuts known and used daily. It closes with satisfaction, future intent, a 0-10 improvement scale, and a full demographics block including age, gender, race/ethnicity, income, education, employment, household, region, locale, computing role, and primary platform.
