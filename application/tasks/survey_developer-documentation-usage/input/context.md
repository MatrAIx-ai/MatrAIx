# Developer Documentation Usage

How developers and technical learners find, read, trust, contribute to, and evaluate technical documentation.

## About this survey

This survey measures documentation habits from first-look sources and preferred formats through the quality attributes people value and the problems that frustrate them. It covers where respondents go first, which reference sites and offline tools they use, how heavily they rely on AI assistants to explain docs, and how they behave when documentation fails. Likert agreement and importance batteries rate the documentation people use most, a ranking item captures the top drivers of usefulness, and satisfaction, recommendation likelihood, and future-reliance questions gauge outcomes and intent. Question types include single-choice, multi-select, several Likert scales, split matrix batteries, a top-3 ranking, and numeric and open free-text, followed by a comprehensive demographic block capturing role, coding experience, age, gender, race/ethnicity, education, employment, income, household, relationship status, region, and area type.
