# Code Editor & IDE Preference

Which code editors and IDEs developers use, how they configure them, and how satisfied they are.

## About this survey

This survey profiles developers' code editor and IDE choices, opening with a screener on coding activity and experience before covering the respondent's primary editor, editors used and wanted, operating systems, languages, and the reasons behind their choice. It measures configuration habits (extensions, keybindings, themes, fonts, terminals, Vim usage) and the features and AI tools respondents rely on. Attitudes are captured with an agreement Likert battery about productivity and customization and a per-item importance matrix over editor-selection criteria, plus a top-three ranking of decision factors. Later sections address performance satisfaction, stability, licensing spend, remote development, overall satisfaction, recommendation likelihood, and switching intent. Question types include single-choice, multi-select, Likert scales, an importance matrix, a ranking item, and numeric free-text. A comprehensive demographic block records role, professional experience, employment, company size, industry, age, gender, race and ethnicity, education, U.S. region, and urbanicity.
