# Pharmacy Loyalty Programs

How consumers enroll in, use, and value retail pharmacy loyalty and rewards programs.

## About this survey

This survey explores consumers' relationships with pharmacy loyalty and rewards programs such as CVS ExtraCare, myWalgreens, and grocery-store pharmacy rewards. It begins with screener questions on pharmacy usage and program enrollment, then measures behaviors including which programs people belong to, how they access and redeem rewards, app feature use, and monthly spending. Likert batteries capture attitudes toward program value, ease of redemption, relevance of offers, and data-privacy concerns, while a top-3 ranking identifies the program features consumers value most. Outcome questions cover satisfaction, estimated savings, likelihood to recommend, and switching intent. A comprehensive demographic block captures age, gender, race and ethnicity, income, education, employment, household size, marital status, U.S. region, community type, and health-insurance status. Question types include single-choice, multi-select, several rating scales, a matrix battery, a ranking item, and numeric free-text.
