# Containers & Docker Usage

How developers use containers and Docker across development, orchestration, and production.

## About this survey

This survey examines container and Docker usage among developers, opening with a screener on frequency, tenure, and the respondent's role with containers. It records the primary container tool and tools used, orchestration platform and Kubernetes distributions, what is containerized, where containers run, registries used, image-build methods, optimization practices, production usage, and learning path. Attitudes are gathered through an agreement Likert battery on workflow, complexity, and security and a per-item importance matrix over container-tooling criteria, plus a top-three ranking of container benefits and a multi-select of challenges. Later sections address blocking problems, image scanning, Docker Desktop licensing, networking comfort, persistent storage, and satisfaction with tooling and performance, followed by overall satisfaction, recommendation likelihood, usage-growth intent, and Kubernetes adoption intent. Question types include single-choice, multi-select, Likert scales, an importance matrix, a ranking item, and numeric free-text. The survey closes with a demographic block covering role, professional experience, employment, company size, industry, age, gender, race and ethnicity, education, U.S. region, and urbanicity.
