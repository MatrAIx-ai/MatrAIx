# Webcam Usage

How people own and use webcams — device types and brands, video-call and streaming habits, purchase criteria, privacy behavior, and satisfaction.

## About this survey

This survey profiles webcam ownership and video-call behavior. After a usage screener, it captures webcam type (built-in, external, phone, dedicated camera) and brand, primary uses (work meetings, personal calls, classes, streaming, telehealth), weekly call volume, platforms used, frequency, camera-on habits, and resolution. Importance batteries rate image quality, low-light performance, microphone quality, price, autofocus, and privacy shutter, plus a top-3 features ranking. Agreement batteries probe whether a good webcam matters, hacking concern, on-camera comfort, built-in quality sufficiency, and appearance/background awareness, while further questions cover webcam covering, call enhancements (virtual backgrounds, lighting, external mics), frustrations, technical problems, usage change over time, satisfaction, purchase intent, and a 0–10 recommendation score. Question types include single-choice, multi-select, importance/agreement/frequency/satisfaction scales, and numeric and open free-text. Demographics include age, gender, race/ethnicity, income, education, employment, work arrangement, region, urban/rural, and tech comfort.
