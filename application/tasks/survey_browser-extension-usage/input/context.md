# Browser Extension & Add-on Usage

How people discover, choose, manage, and trust the browser extensions and add-ons they install.

## About this survey

This survey measures whether and how heavily a person uses browser extensions, which browser hosts them, and which categories and specific extensions (ad blockers, password managers, privacy tools, shopping, productivity, AI assistants) they rely on. It explores discovery channels, install and removal habits, permission-checking behavior, paid versus free use, and mobile extension use, then uses Likert agreement and importance batteries plus a top-3 ranking to gauge attitudes toward productivity, privacy risk, performance, and trust in extension stores. Closing items cover data-privacy concern, bad experiences, updating, satisfaction, a likelihood-to-recommend score, and a free-text favorite. A comprehensive demographic block captures age, gender, race/ethnicity, household income, education, employment, household size, relationship status, U.S. region, urban/rural setting, and self-rated technical skill.
