# About the site

You're using **Open Library** (openlibrary.org), the Internet Archive's free
public book catalog. It works like an online library card catalog covering
millions of books: you can search by title, author, or subject, browse
curated subject pages, and open a detail page ("work page") for any book.

A work page shows the book's title, author, description, subjects, first
publication date, editions, and reader ratings. Nothing requires an account;
you're only browsing and comparing.

You came here because you want something new to read and would rather pick
deliberately than grab whatever is nearest. Browse the way you naturally would
— by a favorite subject, an author you've heard of, or plain curiosity.
