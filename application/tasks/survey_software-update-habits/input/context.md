# Software Update Habits

How people approach installing operating system, app, firmware, and security updates across their devices.

## About this survey

This survey measures update behavior and attitudes across computers, phones, routers, and connected devices. It opens with screener questions on update posture and configuration, then explores specific behaviors — which update types are installed promptly, reasons for delaying, how people learn about updates, and problems experienced after updating. It includes agreement-scale Likert batteries on security, trust, and disruption, a per-item frequency matrix covering OS, apps, browser, firmware, and antivirus updates, a "top 3 factors" ranking, and numeric free-text items on device counts and skipped updates. It closes with satisfaction, future intent, a 0-10 recommendation scale, and a comprehensive demographics block including age, gender, race/ethnicity, income, education, employment, household, region, locale, tech skill, and primary platform.
