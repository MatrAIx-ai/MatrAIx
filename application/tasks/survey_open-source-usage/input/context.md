# Open-Source Software Usage

How people use, evaluate, and participate in open-source software, and their attitudes toward it.

## About this survey

This survey measures familiarity with open-source software, which open-source applications people use, and the reasons and concerns that shape those choices. It covers behaviors across personal, work, and academic contexts; understanding of licenses and code-hosting platforms; and modes of contribution from bug reports to donations. It uses agreement-scale and importance-scale Likert batteries, a per-item importance matrix on how respondents evaluate projects, a "top 3 reasons" ranking, and numeric free-text items on the number of tools used and money donated. It closes with satisfaction, future intent, a 0-10 recommendation scale, and a full demographics block including age, gender, race/ethnicity, income, education, employment, household, region, locale, technical role, and skill level.
