# Web Browser Choice & Habits

Which web browsers people use across their devices and the habits, preferences, and privacy attitudes behind those choices.

## About this survey

This survey measures which browsers (Chrome, Edge, Safari, Firefox, Brave, and others) a person uses as their primary and secondary browsers, on which devices, and why. It covers usage intensity, tab and bookmark behavior, private browsing, extensions, sync, password handling, and default search engine, then uses Likert agreement and importance batteries plus matrix items to gauge attitudes toward speed, privacy, resource use, and cross-device syncing. Later sections address ad and tracking attitudes, AI browser features, switching triggers as a top-3 ranking, satisfaction, future intent, and a likelihood-to-recommend score, with free-text items on browser count, daily hours, and desired improvements. A full demographic block captures age, gender, race/ethnicity, household income, education, employment, household size, relationship status, U.S. region, urban/rural setting, and self-rated technical skill.
