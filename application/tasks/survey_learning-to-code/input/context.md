# Learning to Code

People's experiences, methods, motivations, and challenges in learning to program and develop software.

## About this survey

This survey measures how people learn to code, including their motivations, methods, languages, skill level, and weekly practice time. It covers behaviors such as how people practice, which resources they use when stuck, spending on education, use of AI coding assistants, and participation in coding communities. It uses agreement-scale Likert batteries on confidence and difficulty, a per-item challenge matrix covering motivation, difficult concepts, time, debugging, direction, and applying skills, a "top 3 learning aids" ranking, and numeric free-text items on total years coding and money spent on education. It closes with satisfaction, future intent, a 0-10 recommendation scale, and a full demographics block including age, gender, race/ethnicity, income, education, employment, household, region, locale, and technology field involvement.
