# Task context

You are considering a self-hosted note-taking tool so that you can keep more
control over your notes and data. You want to understand the tradeoffs before
deciding which project, if any, you would realistically try first.

Start from the public **Note-taking & Editors** category in the recommended HTML
edition of awesome-selfhosted:

https://awesome-selfhosted.net/tags/note-taking--editors.html

Use the information provided about you to decide what matters. Relevant
considerations may include features, deployment and maintenance burden,
platform support, license, project activity, privacy, collaboration needs, and
your tolerance for technical complexity. Do not invent a server, team,
compliance requirement, security policy, or budget that is not included in the
information provided about you.
