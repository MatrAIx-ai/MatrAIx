# File Sync & Transfer Habits

How people keep files synchronized and move files between their devices, and the reliability, conflict, and privacy issues they run into.

## About this survey

This survey captures which sync and transfer tools a person uses (Dropbox, Google Drive, OneDrive, iCloud Drive, Syncthing, Resilio, a NAS, USB, AirDrop, and manual copying), how many devices they sync, and which file types and methods they rely on. It measures selective sync, on-demand files, large-file transfers, conflict frequency and resolution, data loss, version history, and handling of sensitive files, then applies Likert agreement and importance batteries plus a problem-frequency matrix and a top-3 factor ranking to gauge attitudes toward availability, reliability, privacy, control, and confusion. Later items cover privacy concern, tool-switching, satisfaction, and a likelihood-to-recommend score, with free-text on device count and desired improvements. A comprehensive demographic block captures age, gender, race/ethnicity, household income, education, employment, household size, relationship status, U.S. region, urban/rural setting, and self-rated technical skill.
