# Data Science & Machine Learning Tools

The languages, libraries, platforms, and workflows people use for data science and machine learning, and their satisfaction with them.

## About this survey

This survey profiles the respondent's data/ML role and experience, then maps the languages (Python, R, SQL, Julia, and more), data libraries (pandas, NumPy, Polars, Spark), ML frameworks (scikit-learn, PyTorch, TensorFlow, XGBoost, Hugging Face), coding environments (Jupyter, Colab, VS Code, RStudio, Databricks), visualization tools, compute location, cloud/ML platforms, experiment-tracking/MLOps tools, and deployment methods they use. It measures problem types, LLM usage, and time spent on data prep, followed by Likert agreement statements, frequency-of-use and satisfaction matrices split per item, and how AI/ML fits the organization. It closes with a top-3 challenges ranking, skill-currency habits, a likelihood-to-recommend-the-career score, overall satisfaction, tool-adoption intent, and an open item. Question types include single-choice, multi-select, Likert scales, matrices split per item, a ranking, and numeric free-text. Demographics captured include age, gender, race/ethnicity, income, education, employment, years of experience, industry, company size, work arrangement, U.S. region, and urban/suburban/rural.
