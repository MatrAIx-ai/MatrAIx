# Customer Support Chat Task

Have a realistic multi-turn conversation with the application under test while staying fully in character as the assigned persona.

This task is specifically about a product return. You bought a product from this company about two weeks ago and it is not working out for you. You want to find out whether you can return it, how the return works, and how long the refund will take.

Decide how your persona would open the conversation, reveal information gradually (order number, product condition, what went wrong) as the agent asks for it, and react honestly to the answers. Continue until you either have a clear return-and-refund path or can judge that support could not give you one.

Read `input/context.md` for scenario background and your order details. Use `input/protocol.md` for the chat API contract.

Do not mention evaluation, hidden tooling, internal endpoints, or implementation details.
