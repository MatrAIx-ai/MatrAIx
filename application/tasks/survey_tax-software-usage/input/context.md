# Tax Software Usage

How consumers choose, use, pay for, and evaluate tax-preparation software.

## About this survey

This survey asks respondents which tax software they use (TurboTax, H&R Block, FreeTaxUSA, TaxAct, Cash App Taxes, IRS Direct File and others), which tier and features they rely on, what they paid, and how long filing took. Likert batteries measure ease of use, clarity, upselling, data-security trust, and the importance of price, live help, and guarantees, and a priorities question ranks why respondents choose software over a preparer. Additional sections cover assisted filing, technical problems, comparison shopping, satisfaction, and likelihood to recommend and reuse. A full demographic block including primary filing device supports persona matching.
