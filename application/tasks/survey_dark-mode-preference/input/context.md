# Dark Mode & Theme Preference

People's preferences for dark mode, light mode, automatic themes, and related display customizations.

## About this survey

This survey measures which display themes people prefer and use across phones, computers, browsers, and apps, along with how long they have used dark mode and why. It covers behaviors such as automatic day/night switching, blue-light filters, and other visual customizations, plus context-specific preferences for reading and viewing media. It uses agreement-scale Likert batteries on eye strain, aesthetics, and battery, a per-item importance matrix on where dark mode matters most, a "top 3 reasons" ranking, and a numeric free-text item on daily dark-mode screen hours. It closes with satisfaction, future intent, a 0-10 recommendation scale, and a full demographics block including age, gender, race/ethnicity, income, education, employment, household, region, locale, a vision-condition item, and tech skill.
